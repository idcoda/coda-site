Kebijakan Privasi

Efektif:

Kebijakan ini menjelaskan bagaimana idcoda (“kami”) menangani data saat kamu menggunakan aplikasi Coda. Coda didesain offline-first dan tidak memerlukan akun.

Ringkasan

  • Tidak ada data pengguna yang dikumpulkan oleh aplikasi untuk tujuan analitik/iklan.
  • Tidak ada data yang dibagikan ke pihak ketiga oleh kami.
  • Semua task & rencana kamu tersimpan lokal di perangkat.

Data yang Dikumpulkan

Coda tidak mengumpulkan data pribadi, lokasi, kontak, atau aktivitas untuk pelacakan. Kami tidak menanamkan SDK iklan atau SDK analitik pihak ketiga dalam aplikasi.

Data yang Kamu Simpan di Perangkat

Aplikasi menyimpan data berikut secara lokal untuk berfungsi:

  • Task, tag, deadline, dan status
  • Daily plan (time blocks)
  • Scorecard / statistik produktivitas (ringkasan lokal)

Data ini berada di storage aplikasi di perangkatmu dan tidak dikirim ke server kami.

Pembelian Dalam Aplikasi (Jika Kamu Upgrade Pro)

Jika kamu membeli versi Pro, transaksi diproses oleh Google Play. Kami tidak menerima informasi kartu pembayaran kamu. Kami hanya menerima status pembelian/entitlement dari Google Play untuk membuka fitur Pro.

Email Support

Jika kamu menghubungi kami via email, kamu akan memberikan informasi secara sukarela (misalnya alamat email dan isi pesan). Informasi tersebut digunakan hanya untuk menanggapi permintaan bantuan dan peningkatan layanan.

Penyimpanan & Retensi

  • Data produktivitas di aplikasi: tersimpan lokal sampai kamu menghapusnya.
  • Email support: disimpan seperlunya untuk menyelesaikan tiket/permintaan.

Keamanan

Karena data utama tersimpan di perangkat, keamanan juga bergantung pada pengamanan perangkatmu (PIN/biometrik, enkripsi perangkat, dsb).

Penghapusan Data

Kamu dapat menghapus semua data Coda kapan pun. Lihat langkah lengkap di halaman Data deletion.

Anak-anak

Coda tidak ditujukan untuk anak di bawah 13 tahun dan tidak mengumpulkan data dari anak-anak.

Perubahan Kebijakan

Jika ada perubahan material, kami akan memperbarui halaman ini dan memperbarui tanggal efektif.
