Penghapusan Data

Coda menyimpan task dan rencana secara lokal di perangkat. Tidak ada akun dan tidak ada server kami yang menyimpan data produktivitas kamu.

Hapus semua data aplikasi

  1. Buka Settings di Android
  2. Pilih Apps → Coda
  3. Pilih Storage
  4. Tekan Clear storage / Clear data

Hapus dengan uninstall

Menghapus aplikasi (uninstall) akan menghapus data lokal Coda dari perangkat.

Data email support
